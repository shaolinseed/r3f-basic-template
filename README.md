## Getting Started



### Install
  ```sh
  npm i
  #or
  yarn install
  ```

### Run development server
  ```sh
  npm run dev
  #or
  yarn dev
  ```
  Open [http://localhost:9339](http://localhost:9339) with your browser to see the result.

### Run Production Build 
  ```sh
  npm run build
  #or
  yarn build
  ```
